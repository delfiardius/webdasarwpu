Konsep Bilangan Prima

Bilangan prima adalah bilangan bulat positif yang hanya dapat dibagi secara sempurna (tanpa sisa) oleh 1 dan dirinya sendiri. Contoh bilangan prima adalah 2, 3, 5, 7, 11, dst. Beberapa hal penting tentang bilangan prima:

  • Bilangan 2 adalah satu-satunya bilangan prima genap, karena semua bilangan genap lainnya dapat dibagi oleh 2
  • Bilangan 1 bukan bilangan prima, karena hanya memiliki satu faktor (1 itu sendiri)
  • Bilangan prima adalah 'Blok Bangunan' dari semua bilangan lainnya, karena setiap bilangan bulat poitif bisa dipresentasikan sebagai hasil perkalian bilangan prima (ini disebut Faktorisasi Prima

            fun main() {
                println("===Program Cek Bilangan Prima atau Bukan===")
                print("Input Angka: ")
                val input = readLine()?.toIntOrNull()
                if (input != null) {
                    if (isPrime(input)) {
                        println("$input adalah bilangan prima")
                    } else {
                        println("$input bukan bilangan prima")
                    }
                } else {
                    println("Input harus berupa bilangan bulat positif")
                }
                println("===Program Berakhir===")
            }

            fun isPrime(bilangan: Int): Boolean {
                if (bilangan < 2) return false
                for (i in 2 until bilangan) {
                    if (bilangan % i == 0) return false
                }
                return true
            }
